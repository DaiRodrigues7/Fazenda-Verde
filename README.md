# 🌾 Fazenda Verde - Sistema de Gestão Agropecuária

Um sistema web completo, responsivo e intuitivo desenvolvido para facilitar a gestão de propriedades rurais. O sistema permite o controle detalhado de rebanhos e o registro da produção diária, tudo protegido de forma individual para cada usuário.

🔴 **Acesse o sistema funcionando aqui:** [https://fazenda-verde.vercel.app](https://fazenda-verde.vercel.app/login.html)

---

## ✨ Funcionalidades

- **🔒 Autenticação de Usuários:** Cadastro, login e recuperação de senha gerenciados com segurança.
- **📊 Dashboard Interativo:** Visão geral da fazenda com contagem de animais e resumo da produção do dia.
- **🐔 Gestão de Galinhas:** Cadastro de lotes, linhagem e quantidade de aves.
- **🐄 Gestão de Vacas e Bois:** Controle por brinco, raça, categoria, peso e data de entrada.
- **🐴 Gestão de Cavalos:** Registro detalhado incluindo nome, raça, filiação, nascimento e função.
- **🐑 Gestão de Ovelhas:** Controle de ovelhas por ID, raça, tipo de lã e idade.
- **📝 Lançamento Diário:** Registro diário da produção de Leite (L), Ovos (unidades) e Lã (kg).
- **🛡️ Segurança de Dados (RLS):** Cada usuário tem seu próprio banco de dados isolado. É impossível que um fazendeiro veja os dados de outro.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Estilização e Responsividade:** [Tailwind CSS](https://tailwindcss.com/)
- **Backend / Banco de Dados:** [Supabase](https://supabase.com/) (PostgreSQL)
- **Autenticação:** Supabase Auth
- **Hospedagem:** [Vercel](https://vercel.com/)
